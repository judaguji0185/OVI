Propiedades Básicas de CSS Display:block

Muestra un elemento como un elemento de bloque (como "p"). Comienza en una nueva línea y ocupa todo el ancho. Hace que el comportamiento del elemento sea como un bloque.

En la siguiente imagen podemos visualizar como un parrafo aplicando el elemento block

Los elementos block respetan los anchos y altos y las margenes verticales, como se observa en la siguiente imagen los elementos se transforman en elemntos de bloque, siempre empiezan en una nueva línea y ocupan el espacio disponible en la línea.
