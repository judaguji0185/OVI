Propiedades Básicas de CSS Display:inline

Cada elemento HTML tiene un valor de visualización predeterminado según el tipo de elemento que sea. El valor de visualización predeterminado para la mayoría de los elementos es block (bloque) o inline (en línea). La propiedad Display establece el tipo de caja generada por un elemento

El valor inline Muestra un elemento como un elemento en línea (como "span"). Las propiedades de altura y ancho no tendrán efecto. La siguiente imagen muestra la diferencia de utilizar el valor inline y de no utilizarlo

Se puede observar que utilizando inline la caja contenedora utiliza todo su espacio para mostrar el texto, es decir el elemento se renderizará en línea con otros elementos

Elementos inline

Los elementos inline no comienzan en una nueva línea, aparecen en la misma línea que el contenido y las etiquetas. Algunos ejemplos de elementos inline son las etiquetas: "span", "strong" y "img".
